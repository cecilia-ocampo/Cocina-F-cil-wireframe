# MindFlow — Wireframes & Screen Structure

## Descripción
Proyecto de diseño UX/UI enfocado en la creación de wireframes y estructura de pantallas para una aplicación mobile de registro emocional.  
El objetivo fue definir la organización del contenido, jerarquía visual y flujo de navegación antes del diseño de alta fidelidad.

## Problema
Tras definir la arquitectura del producto y la navegación, era necesario validar la estructura de las pantallas para asegurar claridad, coherencia y facilidad de uso.  
Sin wireframes bien definidos, el desarrollo de la interfaz podía generar inconsistencias y retrabajo.

## Solución
Se diseñaron wireframes de baja y media fidelidad para las pantallas principales de la aplicación, priorizando:

- jerarquía de información clara  
- ubicación lógica de componentes  
- navegación simple  
- experiencia mobile-first  

Se estructuraron las pantallas para que el usuario pudiera registrar emociones, revisar su historial y navegar entre secciones de forma intuitiva.

## Resultado
Se obtuvo una base estructural sólida que permitió avanzar al diseño de alta fidelidad con mayor coherencia.  
El proyecto reforzó la importancia de iterar en etapas tempranas y de validar la organización del contenido antes del diseño visual final.
