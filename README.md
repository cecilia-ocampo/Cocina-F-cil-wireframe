# Cocina Fácil — Wireframe App

## Descripción
Diseño de wireframe inicial para una app mobile de recetas simples dirigida a usuarios con poca experiencia en cocina.

## Problema
Usuarios principiantes necesitan una aplicación clara y fácil de usar sin sobrecarga de información.

## Solución
Se diseñó la pantalla de inicio y estructura base de navegación priorizando:
- claridad visual  
- jerarquía simple  
- acciones principales visibles  

Se trabajó desde boceto en papel hasta wireframe digital en Figma.

## Resultado
Se obtuvo una base estructural clara para el desarrollo posterior de la aplicación y comprensión del flujo inicial del usuario.
